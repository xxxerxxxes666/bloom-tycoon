# GPT Style Design Prompt — Bloom Tycoon

Use this prompt with GPT/image generation to expand the selected game visual identity.

```text
Design the art direction for a mobile game called Bloom Tycoon.

Genre: cozy-luxury match-3 + light tycoon.
Theme: building a floral shop from a humble cart into an elegant boutique empire.
Selected aesthetic: Dark elegant floral.
Audience: casual mobile players, slightly more premium and stylish than generic candy games.
Mood: inviting, polished, fragrant, optimistic, tasteful, lightly magical, mysterious but not spooky.

Core palette:
- Black plum background: #120814
- Deep plum surfaces: #211024 and #321837
- Rose gold accent: #d8a657
- Ivory text/flower highlights: #fff7ea and #f3ead7
- Emerald leaf/growth accent: #4f9f79
- Blush highlight: #e8b6c9
- Mauve borders: #5a334f

Create a cohesive style guide including:
1. Final color palette with hex codes.
2. UI style: buttons, panels, level cards, shop upgrade cards.
3. Flower tile designs for rose, tulip, lily, orchid, sunflower.
4. Shop upgrade visual progression:
   - Flower Cart
   - Bouquet Table
   - Display Shelf
   - Greenhouse Corner
   - Luxury Counter
5. Typography suggestions.
6. Icon style.
7. App icon concept.
8. App Store screenshot direction.
9. Small animation/motion ideas for matching and upgrades.
10. What to avoid visually.

Important constraints:
- Keep it readable and warm for casual mobile players.
- Avoid Halloween/goth clichés.
- Avoid generic rainbow candy-game visuals.
- Final tiles must be readable on a phone screen.
- The game should feel like a luxury floral boutique at dusk: velvet plum, rose-gold signage, ivory flowers, emerald leaves, soft candle glow.

Return the answer as a practical game art direction document that a UI artist or AI image generator can use.
```

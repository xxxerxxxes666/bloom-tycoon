# Bloom Tycoon — Name Research & Recommendation

## What changed

`Flower Empire` / `Bloom Empire` is too broad. It says “scale,” but not the actual play promise. The name should instantly tell a casual player:

> flowers + progression/business growth.

## App Store scan summary

A quick Apple iTunes Search API scan showed:

- `Bloom` and `Flower` are crowded keywords.
- `Bloom Empire` did **not** appear as an exact obvious direct competitor in top returned results, but it feels generic.
- `Bloom Tycoon` returned no obvious exact game-title competitor in the top results; results skewed toward unrelated tycoon/investing apps.
- `Flower Shop Tycoon` is clear but longer and more generic; nearby results include `Flower Shop Idle` and `The Cozy Florist`.
- `Garden Match Tycoon` / `Flower Match Tycoon` are descriptive but clunky.

## Candidate ranking

| Name | Verdict | Why |
|---|---|---|
| **Bloom Tycoon** | **Best working title** | Short, simple, instantly defines flowers + tycoon progression; less crowded than generic flower/bloom puzzle names. |
| **Flower Shop Tycoon** | Clear but plain | Very literal; useful subtitle/ASO phrase, but not a premium brand name. |
| **Bouquet Tycoon** | Distinct but narrower | Good business fantasy, less immediately garden/flower broad. |
| **Garden Tycoon** | Too broad | Clear, but could mean farming/landscaping rather than floral match. |
| **Petal Tycoon** | Cute but weaker | Less instantly clear than Bloom Tycoon. |
| **Midnight Bloom** | Strong style name | Excellent occult boutique flavor, but does not instantly define tycoon gameplay. |
| **Arcana Bloom** | Strong occult name | Distinctive, but less commercially obvious for casual users. |

## Recommendation

Use:

# **Bloom Tycoon**

Optional subtitle / App Store phrasing:

> **Bloom Tycoon: Garden Match**

This gives us:

- clear category promise,
- simple memory hook,
- useful App Store keywords,
- room for occult/gothic boutique flavor in art and copy without making the title too niche.

## Positioning line

> Match flowers in a moonlit garden, then grow an occult luxury boutique into a floral empire.

## Naming note

Keep `Midnight Bloom` as a chapter/event/style phrase — e.g. `Midnight Bloom Gala`, `Midnight Bloom Pass`, `Midnight Bloom Garden` — but the main title should stay more instantly legible for broad casual acquisition.

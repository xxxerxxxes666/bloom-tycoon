# Bloom Tycoon — Feature Roadmap Borrowed from the Greats

## Guiding principle

Copy proven casual-game structure, then reskin the emotion into **dark elegant floral boutique empire**.

## Milestone 1 — Core puzzle satisfaction

- [x] Match-4 line special: **Rose Shears** readability/naming foundation
- [x] Match-5 color special: **Golden Lily** readability/naming foundation
- [x] T/L area special: **Orchid Burst** 3x3 clear foundation
- [x] Basic special swap activation: specials can fire when swapped even without a normal match
- [x] Advanced combo foundation: Golden + Golden special swaps clear the full board
- [x] Line + area advanced combo: **Shears Burst** clears three rows and three columns
- [ ] Additional advanced combo interactions: line + line, color + line/area
- [x] Win/loss modal foundation with near-win remaining-objective copy
- [ ] Fast retry

Why: Candy Crush and Royal Match live or die by special-piece tactility.

## Milestone 2 — Progression stickiness

- [x] Save/load coins, level, upgrades, Daily Bloom, and spooky-luxury temporary effects
- [x] Next-upgrade progress meter/helper
- [x] Upgrade reveal ceremony foundation
- [ ] 10-level beginner sequence
- [ ] First chapter completion
- [ ] Locked next room tease

Why: Gardenscapes/Homescapes prove that meta progression turns puzzle wins into emotional investment.

## Milestone 3 — Boutique identity

- [ ] Replace placeholder tiles with final dark elegant floral assets
- [ ] Shop background illustration stages
- [ ] App icon polish
- [ ] Sound palette: soft chime, petal pop, brass bell
- [ ] Luxury UI pass

Why: The market has enough match-3. The identity has to carry the first impression.

## Milestone 4 — Retention systems

- [x] Daily reward foundation: **Daily Bloom** streak and coin reward
- [x] Spooky-luxury temporary gameplay effects: **Candlelit Focus** and **Midnight Velvet**
- [x] Reward chest ceremony foundation: **Velvet Bloom Chest** win copy
- [ ] Streak track
- [x] Floral collection album skeleton: **Flowerpedia** lore/readability cards
- [ ] Persisted Flowerpedia unlocks and collection rewards
- [ ] First limited-time event shell

Why: Top grossing casual games create reasons to return outside the current level.

## Milestone 5 — Monetization-ready prototype

- [ ] Booster inventory
- [ ] Extra moves offer placeholder
- [ ] Starter pack placeholder
- [ ] Cosmetics catalog placeholder
- [ ] Event pass placeholder
- [ ] Analytics event names documented

Why: We need monetization architecture without prematurely becoming pushy.

## Milestone 6 — Mobile vertical slice

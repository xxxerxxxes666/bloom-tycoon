# Bloom Tycoon — Occult Gothic Boutique Visual Direction

## Updated creative direction

Xerxes selected: **occult / gothic / tarot / metaphysics / boutique gentle satanism**.

The direction should feel like:

> A nostalgic old casual game discovered in a velvet-lined cabinet: tarot cards, black roses, moonlit conservatories, brass scissors, candle smoke, Latin engravings, and boutique floral luxury.

## Style pillars

1. **Boutique occult, not horror**
   - No cheap gore, jump scares, skull spam, or Halloween party kitsch.
   - Use candlelight, veils, antique cards, sigils, moons, black marble, brass, velvet, and rare blooms.

2. **Nostalgic old-game charm**
   - Think lush 2000s hidden-object / puzzle-game richness, but cleaner for mobile.
   - Painted backgrounds, ornate frames, readable icons, tactile UI.
   - Avoid sterile vector slop and over-smoothed AI rendering.

3. **Tarot/metaphysics as structure**
   - Levels/orders can be grouped by “arcana chapters.”
   - Boosters can use tarot-esque naming: The Shears, The Moon, The Empress, The Tower.
   - Flowerpedia can become an herbarium/grimoire.

4. **Gentle Solve et Coagula energy**
   - Use Latin subtly: `Solve et Coagula`, `Lux Noctis`, `Sub Rosa`, `Flos Umbrae`.
   - The feel is esoteric and elegant, not edgy teenage satanism.

## Visual tokens

| Element | Direction |
|---|---|
| Background | Black plum, midnight violet, smoky gradients, candle glow |
| Accent | Tarnished gold / rose gold / brass |
| UI frames | Tarot-card borders, antique brass, black marble, velvet panels |
| Tiles | Jewelled botanical tokens with symbols, not generic flowers |
| Special tiles | Occult tools: shears, moon seal, candle, sigil, star |
| Rewards | Velvet chest, wax seal, card reveal, flower bloom under moonlight |
| Progress | Ritual circle / moon phase / gilded bar |

## AI art curation rules

Reject generated art if it has:

- Mushy AI flowers with unclear silhouettes.
- Random illegible symbols pretending to be Latin.
- Overly glossy generic fantasy UI.
- Too much purple neon.
- Horror clichés that hurt casual appeal.
- Inconsistent lighting/perspective between assets.
- Unreadable small icons.

Accept only if:

- It reads clearly at phone size.
- It feels handcrafted or painted.
- It has cohesive palette and material logic.
- It supports gameplay readability first.
- It could plausibly belong in a premium boutique puzzle game.
